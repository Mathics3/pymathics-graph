Installation
-------------

From pip:

::

   $ pip install pymathics-graph

From git:

::

   $ make develop  # or make install

Note:
-----

Currently this works well in `mathicsscript` but not in the Django interface, although graphs are created in a temporary directory, e.g. ``/tmp/``.
